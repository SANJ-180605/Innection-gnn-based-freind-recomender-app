# Inection - Interest-Based Connection Platform

A full-stack social platform where users connect based on shared interests, hobbies, and activities.

## 🚀 Features

- User authentication (Signup/Login)
- Interest-based user discovery
- Create and share posts
- Connect with like-minded people
- Real-time messaging
- Personalized recommendations using GNN

## 🛠️ Tech Stack

### Backend
- Node.js / Express
- MongoDB with Mongoose
- JWT Authentication
- Graph Neural Networks (GNN) for recommendations

### Frontend
- React.js
- React Router
- CSS3 with custom styling
- Framer Motion animations

## 📋 Prerequisites

- Node.js (v14 or higher)
- MongoDB (local or Atlas)
- npm or yarn
